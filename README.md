# DSA Visualizer

A terminal-based Python application that visualizes Data Structures and Algorithms step by step. The project is designed to help students understand how algorithms work through interactive execution, detailed comparisons, and rich terminal output.

## Features

- Bubble Sort Visualization
- Binary Search Visualization
- Step-by-step execution
- Comparison and swap statistics
- Rich terminal interface

## Tech Stack

- Python
- Rich

## Planned Features

- Quick Sort
- BFS & DFS Visualization
- Tree Algorithms
- Streamlit Web Interface
